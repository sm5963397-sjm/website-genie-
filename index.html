◈ Summon the GENIE

Quick Install

npx genie-setup init
$genie scan ./src# deep AST analysis
$genie repair --plan# generate fix plan
$genie apply --approve# apply patches
$genie test# validate fixes
$genie docs --format md# generate docs

Requires Node 18+ · Works on Windows · macOS · Linux

◈ The Ritual

How It Works

01
genie scan

Full AST traversal. Identifies dead functions, circular deps, and calculates entropy scores across every file.

02
genie repair --plan

AI agents orchestrate targeted repair strategies. Multiple models propose solutions, GENIE picks the best.

03
genie apply --approve

Review diffs, approve changes individually or in batch. Full git integration — never committed blindly.

04
genie test && genie docs

Validate patches with automated testing, then regenerate living documentation. Entropy score recalculated.
